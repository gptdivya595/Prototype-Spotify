# Discovery Evidence Lab

Discovery Evidence Lab is the user-facing name for **Artifact A**, the evidence system that turns
public Spotify feedback into a traceable problem hypothesis and an Artifact B experiment.

The `artifact A` folder name stays unchanged for command and link compatibility. **Solutioning**
is the decision process, and **Discovery Compass** is the proposed Artifact B product.
